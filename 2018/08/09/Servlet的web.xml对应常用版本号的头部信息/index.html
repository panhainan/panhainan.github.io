Servlet的web.xml对应常用版本号的头部信息

2018/08/09 Servlet web.xml版本号

本文于 435 天之前发表，文中内容可能已经过时。

这篇笔记提供Servlet目前常用的2.5，3.0以及3.1版本对应的web.xml的头部信息。

Servlet 2.5

deployment descriptor Java EE 5 XML schema, namespace is http://java.sun.com/xml/ns/javaee

Servlet 3.0

deployment descriptor Java EE 6 XML schema, namespace is http://java.sun.com/xml/ns/javaee

Servlet 3.1

deployment descriptor Java EE 7 XML schema, namespace is http://xmlns.jcp.org/xml/ns/javaee/
